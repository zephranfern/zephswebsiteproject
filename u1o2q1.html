Unit 1 Outcome 2

1. How useful were the tools that I used?

To plan my website I used trello to detail the questions and steps I had to overcome. It was very useful to have all of my objectives in easy to understand areas and it was simple to create and move each objective to the appropriate list. To physically create my website I used Visual Studio Code. Visual Studio Code itself was easy to use and understand. It allowed me to push all my content to GitHub. Speaking of GitHub, once I understood the steps I had to take while using the terminal in Visual Studio Code, it was simple to upload all of my content and make updates and create save files for my code.
