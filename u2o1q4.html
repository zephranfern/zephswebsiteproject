Unit 2 Outcome 1

4. A description of the appropriate filetypes for websites.

Depending on what it is you're wishing to upload will change the file type you use. For uploading websites you will most likely want to use HTML as it’s what most websites use and it is relatively easy to use. For uploading photos you will probably want to use JPEG, PNG or GIF. You’ll use JPEG for things like photos, you’ll use PNG for images used in websites as they sport many different features and you’ll us GIF for animated images used in websites as, like PNG, they sport many different features but they can be animated as well. However GIF files are larger than PNG files. For more information on file types visit https://www.yourhtmlsource.com/starthere/fileformats.html.
