1. How efficient were the tools and systems used in the project?

I would say that the tools used in this project were efficient enough for this project. Visual Studio Code came up with options for different syntax I could use as I was typing without obscuring the rest of the code too much and it auto indented certain areas of code making it easier to read and understand. Trello was efficient in how it portrayed and stored each objective. GitHub, once you understood the steps needed, quickly and safely uploaded and stored code in its entirety. I also found that using a Mac made the process faster and smoother than using a standard intel device as the Mac is a much more powerful device.
