5. How I have met my brief.

I started by preparing my trello board. Then using pages on the Mac I researched and drafted all the content to my site. Once I was happy with the content I copied it and pasted it into Visual studio Code and started organising it inside HTML. I then started using CSS to implement a navigation bar so that I could easily switch between the information. I then added my links where necessary to each of my information pages I then added several images to the website to make it more visually appealing. I had to change some of the image file types from JPG to PNG to insure no errors would ensue later. I then tested the code I had written to ensure the website was functional.
